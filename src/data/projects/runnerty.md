Runnerty es un motor ligero y flexible de orquestación de trabajos (job orchestrator) diseñado para automatizar y programar tareas y flujos de trabajo en entornos distribuidos. Está pensado para ser sencillo de configurar y extender, y ofrece:

- **Definición de workflows** mediante archivos de configuración en JSON o YAML.
- **Ejecución y reintentos** automáticos con control de errores y timeouts.
- **Conectores y plugins** para integrar con colas, bases de datos y servicios externos.
- **Monitoreo y logging** centralizado para observar el estado de las ejecuciones.

Se emplea para orquestar tareas periódicas o dependientes (ETL, pipelines de datos, despliegues), aportando una solución modular y menos pesada que alternativas empresariales.

Más información: https://runnerty.io/
